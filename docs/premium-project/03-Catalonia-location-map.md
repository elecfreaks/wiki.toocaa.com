# 加泰罗尼亚地图
## 作品描述
这是一款互动式拼图，以加泰罗尼亚地区地图为主题，包含多个独立拼块，每块对应不同的城市或区域。激光雕刻清晰展现地理边界和主要城市标识，让拼图不仅具备拼接的趣味性，还能作为教育工具或展示品。适合旅行爱好者、学生、文化研究者，可以边拼边了解这片区域的地理分布。<br/>
