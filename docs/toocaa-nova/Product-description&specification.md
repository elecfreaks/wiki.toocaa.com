# 了解您的 TOOCAA NOVA

**TOOCAA NOVA** 是一款紧凑型桌面激光雕刻系统，旨在让数字制造更加安全、智能且易于上手。通过 **模块化设计**、**智能自动化** 与 **认证级安全防护**，TOOCAA NOVA 有效解决了传统开放式雕刻机在复杂度与安全性方面的局限。

可更换式激光模组支持 **2W IR、10W 与 20W** 多种功率配置，能够灵活适配不同材料，延长产品使用寿命并减少资源浪费。全封闭 **Class 1 安全外壳** 内置火焰检测、倾斜检测与儿童锁，确保设备在各种使用场景下安全可靠。**双通道智能气流系统** 有效保持腔体清洁，并提升加工稳定性与精度。

由 AI 驱动的 **TOOCAA Studio** 软件可自动完成图像生成、材料识别与参数推荐，大幅降低新手学习门槛。配合多语言界面与多格式兼容能力，TOOCAA NOVA 为个人创作者与专业用户提供了一套 **安全、智能、可持续** 的创作解决方案。

# 设备连接操作（Device Connection Instructions）

1. 开箱后，从机身内置泡棉中取出 **激光模组、激光模组螺丝与扳手**  
   > 注：螺丝与扳手位于 ToolBox 内
2. 将激光模组沿 X 轴底槽插入，并使用螺丝固定
3. 取出电源适配器与电源线，按指示方向正确连接
4. 取出气泵、气管与 XT30 线
5. 将气管与 XT30 线分别连接至气泵与整机
6. 插入风管并逆时针旋转，直至锁定
7. 确认限位器位置，防止耗材偏移

## TOOCAA NOVA & TOOCAA Studio 使用教程

1. 启动 **TOOCAA NOVA**，并使用 Google Chrome 打开 **TOOCAA Studio**  
   👉 https://www.toocaastudio.com/
2. 点击界面右上角 **连接设备**
3. 选择 **TOOCAA NOVA**，点击 **Confirm**
4. 选择连接方式（USB / Wi-Fi）

---

### USB 连接

1. 使用 USB Type-C 数据线连接 NOVA 与电脑  
2. 在设备连接弹窗中点击 **Connect**
3. 在浏览器串口选择窗口中选择对应设备

---

### Wi-Fi 连接

> 请确保 NOVA 与电脑连接至 **同一 Wi-Fi 网络，并处于同一 IP 网段**

1. 在 NOVA 触摸屏进入 **设置 → Wi-Fi**
2. 选择 Wi-Fi 并输入密码
3. 在 TOOCAA Studio 中选择 **Wi-Fi 连接 → 连接新设备**
4. 输入 NOVA 的 **IP 地址** 并点击 **Connect**
5. 在 NOVA 触摸屏上点击绿色 **确认** 按钮完成连接

连接成功后，TOOCAA Studio 将显示当前已连接设备（激光模组、气泵、旋转配件等）。

## TOOCAA Studio 创作教程

1. 打开提供的项目文件（File → Open）
   http://wiki-toocaa.oss-cn-hongkong.aliyuncs.com/NOVA/Lucky%20Koi!.TCS
3. 放置 **耗材识别卡**，自动完成材料识别
4. 确认参数推荐状态
5. 移除识别卡，点击 **Frame** 确认加工位置  
   > 走边框期间请勿开盖
6. 点击 **Auto Focus**（或 Advanced Focus）
7. 点击 **Start Processing** 开始加工
8. 可修改文件名后点击 **Start**

加工完成后，你的第一个作品即告完成。

---

## TOOCAA NOVA 离线创作教程

1. 在触摸屏选择 **本地文件模式（Offline Mode）**
2. 选择 **Lucky Koi!** 文件
3. 放置耗材并点击 **Next**
4. 执行 **Frame** 与 **Auto Focus**
5. 确认加工内容与时间，点击绿色 **Start**
6. 加工完成，作品生成
