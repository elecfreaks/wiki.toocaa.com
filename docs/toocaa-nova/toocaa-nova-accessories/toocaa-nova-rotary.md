# TOOCAA NOVA旋转配件
# 了解您的TOOCAA NOVA旋转配件

TOOCAA NOVA 旋转配件是一款多功能旋转配件，专为各种材料的精密加工而设计。  
它提供 **2 种配件类型**，可在多种场景下处理不同类型的材料，满足追求高质量雕刻与切割解决方案的爱好者与专业人士需求。  
适用于：**手工项目、模型制作、小批量生产及教育用途**。

## 核心特点

- **【灵活百搭】**  
  2 合 1 多功能旋转配件，180° 可调三爪卡盘适用于约 90% 的日常圆柱体、球体及各类异形物体，可在酒杯、珠宝、棒球棒、戒指等材料上自由创作。

- **【独家专利】**  
  全球独有的三秒快拆钳口专利设计，通过快速更换不同类型的锁紧部件，实现不同直径物体的 360° 雕刻，无需拆卸与收纳零件，大幅提升创作效率。

- **【极速精准】**  
  激光旋转卡盘最大雕刻速度可达 360°/秒，配备高精密步进电机，最小步距角度 0.02°，精准还原创意细节。

- **【服务至上】**  
  使用过程中如遇任何问题，请随时联系我们。TOOCAA 产品专家将在 24 小时内提供专业技术支持与贴心服务。

## TOOCAA Studio & TOOCAA NOVA Rotary 使用教程

1. 启动 TOOCAA NOVA，并打开 **TOOCAA Studio**。  
   正确连接设备后，界面将显示当前已连接的配件。

  <img src="http://wiki-toocaa.oss-cn-hongkong.aliyuncs.com/NOVA/Rotary/17.jpg" alt="图片描述" width="1000" /><br/>

2. 输入 **物体直径**。

  <img src="http://wiki-toocaa.oss-cn-hongkong.aliyuncs.com/NOVA/Rotary/1.jpg" alt="图片描述" width="1000" /><br/>

3. 在画布内完成创作后，执行 **走边框（Frame）** 操作，确认加工区域位于耗材上，避免激光打穿机器。

4. 根据材料硬度选择对焦方式：  
   - 硬质材料：自动对焦  
   - 软质材料：手动调整激光头高度进行对焦

5. 对焦完成后，即可开始加工。

## 常见问题解答（FAQ）

### 1. 旋转配件不旋转或旋转不规律

1. 确保机器与 X/Z 轴连接正确。  
2. 检查 TOOCAA Studio 中填写的 **耗材直径参数** 是否正确。  
3. 若确认连接与参数均无误仍异常，请联系 TOOCAA 售后技术支持：  
   **support@elecfreaks.com**

---

### 2. 拓展夹爪 / 卡盘卡针无法插入
