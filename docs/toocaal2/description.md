## 报警说明
当不器 L2触发报警状态时，会自动关闭激光输出。

| **报警声** | **含义** |
| :---: | :---: |
| 短响1声 | 触发限位开关 |
| 短响2声 | 激光加工过程中打开防护罩 |
| 短响3声 | 未检测到USB锁 |
| 短响4声 | 未接入激光模组 |
| 长响1声，短响1声 | 检测到机器倾斜 |
| 长响1声，短响2声 | 电源不匹配 |
| 长响1声，短响3声 | 激光加工过程中TF卡出现异常 |
| 长响1声，短响4声 | 工作区域检测到火焰 |
